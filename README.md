# Secure Online Payment System - Pseudocode

Pseudocode for securing an online payment system against SQL injection 
and cross-site scripting (XSS) attacks.

## Course
MSCS-535 Secure Software Development  
University of the Cumberlands

## Files
- [secure_payment.md](secure_payment.md) - Main payment processing flow with HTTPS enforcement and session validation
- [sql_injection_defense.md](sql_injection_defense.md) - Parameterized query defense against SQL injection
- [xss_defense.md](xss_defense.md) - Input sanitization and output encoding defense against XSS

## References
- Richardson, T., & Thies, C. N. (2012). Secure software design. Jones & Bartlett Learning.
- Kohnfelder, L. (2021). Designing secure software. Random House Publishing Services.
